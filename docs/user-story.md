# Tableau User Story | Canadian Substance Use by Age Group

## Introduction
This dashboard explores how alcohol, cannabis, and opioid 
use differs across age groups in Canada, using 2023 survey 
data from Health Canada. The goal is to help any Canadian 
understand at a glance whether substance use is really a 
"young people problem" or whether the patterns are more 
surprising and complex than that.

## Dashboard Requirements

### Dashboard Purpose
To show how substance use rates for alcohol, cannabis, and 
opioids vary across four Canadian age groups: 15-19, 20-24, 
25-54, and 55+, using percentage of population data from 
the Canadian Substance Use Survey 2023.

### Key Requirements

**Substance comparison by age group**
Show what percentage of each age group has used alcohol, 
cannabis, and opioids, so we can see which substances 
dominate which age group at a single glance.
Chart type: Grouped bar chart (one group per age bracket, 
3 colored bars per group)

**Binge drinking deep dive**
Show which age group binge drinks the most in Canada.
Chart type: Horizontal bar chart, sorted by percentage

**Opioid misuse deep dive**
Show which age group uses opioids for non-medical purposes.
Chart type: Horizontal bar chart, sorted by percentage

### Design and Interactivity Requirements

**Filters**
Allow the user to filter by age group so they can focus 
on one generation at a time.

**Tooltips**
Every bar shows exact percentage and sample size on hover 
so non-technical readers understand what the number means.

**Color coding**
Blue for Alcohol, Green for Cannabis, Amber for Opioids.
Consistent across all 3 charts.

# Canada Substance Use Dashboard

## What is this project?
This project explores how alcohol, cannabis, and opioid use 
varies across different age groups in Canada, using real 
survey data collected by Health Canada.

## Why did I build this?
I wanted to understand whether substance use patterns 
really do differ across generations in Canada, and if so, 
how dramatically. The data tells a surprisingly compelling 
story that I think every Canadian should see.

## Data Source
Canadian Substance Use Survey (CSUS) from Health Canada
health-infobase.canada.ca/substance-use/csus

## Tools Used
- SQL for data cleaning and preparation
- Tableau for visualization and dashboard building

## Project Structure
- data/ — raw CSV files from Health Canada
- sql/ — all SQL queries used to clean and prepare the data
- docs/ — project documentation and writeup
